# E-Commerce Data Insights

## Objective
Analyze e-commerce sales data to derive insights for business growth.

## Tools Used
Python, Pandas, Matplotlib

## Features
- Top-selling products analysis
- Peak shopping hours detection
- User retention analysis
- Data visualization

## How to Run
1. Install required libraries
2. Run ecommerce_analysis.py
3. View charts and insights

## Outcome
Helps businesses improve stock planning and user engagement.
